# Roadmap

## MVP 0.1

- Estructura React + Tauri v2 + Rust.
- SQLite con schema, indices, FTS5 y settings.
- UI compacta con busqueda, tabs, lista virtualizada y dialogs.
- Comandos para historial, favoritos, pineados y colecciones.
- Atajo global `Ctrl + Alt + V`.
- Captura local de texto del portapapeles con polling conservador.
- Workflow de release para Windows, macOS y Linux.
- README y archivos open source.

## Beta 0.2

- Soporte de imagenes con thumbnails.
- Soporte de documentos como referencias locales.
- Menu completo de tres puntos.
- Preferencias de tema, limite de historial y shortcut.
- System tray opcional.

## Beta 0.3

- Navegacion completa por teclado.
- Accesibilidad.
- Pruebas unitarias Rust.
- Pruebas de UI con Playwright o WebDriver.
- Optimización de memoria para historiales grandes.

## Stable 1.0

- Builds firmados para Windows, macOS y Linux.
- Instaladores nativos.
- Documentacion de arquitectura final.
- Politica de seguridad y releases.
- Capturas oficiales y demo.
