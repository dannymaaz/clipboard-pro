# Security Policy

## Privacy Model

Clipboard Pro is local-first:

- No telemetry.
- No analytics.
- No cloud sync.
- No external services.
- No account system.

Clipboard contents remain on the user's machine.

## Reporting a Vulnerability

Please open a private security advisory on GitHub when the repository is public, or contact the maintainers directly before public disclosure.

Include:

- Affected version.
- Platform.
- Reproduction steps.
- Expected and actual behavior.
- Potential impact.

## Scope

In scope:

- Local data exposure.
- Unsafe file handling.
- Clipboard leakage.
- Dependency vulnerabilities.
- Tauri command permission issues.

Out of scope:

- Social engineering.
- Issues requiring physical access without an app flaw.
- Vulnerabilities in unsupported operating system versions.
